# Student Performance Analysis

## Overview

This project analyzes student academic performance using Python, Pandas, NumPy, and Power BI. The objective is to identify the key factors influencing student success by examining attendance, study hours, demographics, and examination scores.

The project includes data cleaning, exploratory data analysis (EDA), statistical analysis, and interactive dashboard creation to help educators make data-driven decisions and improve student outcomes.

---

## Objectives

* Analyze student academic performance data.
* Identify factors affecting student scores.
* Study the relationship between attendance, study hours, and academic results.
* Discover trends and patterns in student performance.
* Build interactive dashboards for performance monitoring.
* Generate actionable insights for educators and institutions.

---

## Dataset

The dataset contains 500+ student records with information such as:

* Student ID
* Gender
* Age
* Attendance Percentage
* Study Hours
* Exam Scores
* Subject Marks
* Academic Performance Indicators

## Key Findings

### Major Insights

* Students with attendance below 75% showed a significant decline in average scores.
* Higher study hours were positively correlated with improved academic performance.
* Attendance and study consistency emerged as strong indicators of student success.
* Early identification of at-risk students can improve intervention outcomes.

## How to Run

### Clone Repository

```bash
git clone https://github.com/your-username/student-performance-analysis.git
```

### Install Dependencies

```bash
pip install pandas numpy matplotlib
```

### Run the Project

1. Open the project folder in VS Code.
2. Open the terminal.
3. Run the Python script:

```bash
python student_analysis.py
```

4. Review generated visualizations and insights.
5. Open the Power BI dashboard file (.pbix) for interactive analysis.
